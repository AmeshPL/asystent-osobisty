# Asystent personalny

## Uruchomienie lokalne

1. `cd backend`
2. `npm install`
3. Skopiuj `.env.example` do `.env` i wklej swoj `ANTHROPIC_API_KEY`.
4. `npm start`
5. Otworz `http://localhost:3000` w Chrome.

Rozpoznawanie i synteza mowy dzialaja w przegladarce (Web Speech API) - najlepiej wspierane w Chrome na Androidzie.
